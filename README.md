# Easy-Shopping
<br>
Primeiro Commit responsivo.
<br>

### Descrição
O Easy-Shopping é uma plataforma que facilita a experiência de compras, conectando usuários a lojas e produtos de forma prática e eficiente. Nosso objetivo é otimizar o processo de compra, garantindo conveniência e satisfação.
<br>
### Funcionalidades
Busca Inteligente: Localize produtos rapidamente por meio de filtros e categorias personalizadas.
<br>
Comparação de Preços: Compare os valores de um mesmo produto em diferentes lojas para encontrar a melhor oferta.
<br>
Carrinho de Compras: Adicione itens ao carrinho e acompanhe o resumo total das suas compras.
<br>
Avaliações e Feedback: Verifique opiniões e avaliações de outros usuários para ajudar na decisão de compra.
<br>
Integração com Lojas Parceiras: Conecte-se diretamente a lojas confiáveis e seguras.
<br>
Design Responsivo: Interface otimizada para dispositivos móveis e desktops, garantindo uma experiência de compra fluida.

### Tecnologias Utilizadas
- **HTML5**
- **CSS**
